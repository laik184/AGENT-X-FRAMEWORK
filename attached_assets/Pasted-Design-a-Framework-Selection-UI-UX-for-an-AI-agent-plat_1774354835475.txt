Design a Framework Selection UI/UX for an AI agent platform.

STRICT RULE:
- Do NOT change any other part of the app
- Do NOT modify existing layout, navbar, sidebar, colors, or UI
- Only design the framework selection and framework-specific pages
- When a framework button is clicked, ONLY the corresponding framework page should open
- No other UI or page should be changed, edited, or redesigned

CORE FLOW:
- User clicks "Create Project" or "Templates"
- Framework selection screen opens
- User selects a category (Android, iOS, Website, Backend)
- Then selects a framework
- Clicking a framework button MUST open its dedicated setup/start page

STYLE:
- Match existing app design (no full redesign)
- Dark theme, minimal, clean SaaS feel
- Rounded cards (12-16px)
- Subtle glow and hover animations

---

MAIN SCREEN (CATEGORY SELECTION):

Show 4 main cards:
1. Android
2. iOS
3. Website
4. Backend

Each card:
- Icon + title + short description
- Hover glow effect
- Click → opens framework list page ONLY
